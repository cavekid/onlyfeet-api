Welcome to the OnlyFeet SAFE ✨ Project

Goals

    Keep the platform safe from adult content
    Keep the platform safe from filthy individuals1
    Keep the platform safe from anything that is considered inappropriate

Trivia

Basically, we came to the conclusion that ultimately everyone's job can be replaced by AI ✨ in the future. That's why we eventually stopped searching and hiring Content Security Administrators but decided to develop SAFE AI ✨ (Safety for Average Foot Enjoyers) instead - an automated content filter specifically tailored to our platform needs. And because everyone is doing it nowadays, we're doing it with AI! ✨

Contributing

We are still in early development. But our AI Engineers told us you can all help the project already by contributing labeled images so we can train SAFE AI ✨ on what is appropriate content and what is not. You can use our public API for that.

To get started, read the OpenAPI Documentation
